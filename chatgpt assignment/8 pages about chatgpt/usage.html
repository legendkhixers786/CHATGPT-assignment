How to Use ChatGPT

Using ChatGPT is simple and straightforward. Follow these steps to get started:

  1. Obtain an API key from the OpenAI platform.
  2. Integrate the ChatGPT API into your application or platform.
  3. Send requests to the API endpoints with your input text.
  4. Receive AI-generated responses in real-time.

Example Usage

const fetch = require('node-fetch');

async function getChatGPTResponse(inputText) {
    const response = await fetch('https://api.openai.com/v1/engines/text-davinci-003/completions', {
        method: 'POST',
        headers: {
            'Content-Type': 'application/json',
            'Authorization': 'Bearer YOUR_API_KEY_HERE'
        },
        body: JSON.stringify({
            prompt: inputText,
            max_tokens: 100
        })
    });
    const data = await response.json();
    return data.choices[0].text.trim();
}

// Example usage
const inputText = 'How does ChatGPT work?';
getChatGPTResponse(inputText).then(response => {
    console.log(response);
});
